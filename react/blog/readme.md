# Blog App

Reactを利用し、実装したブログアプリケーションです。

---

## 開発環境

- Node.js
- npm
- React

---

## セットアップ

まず、プロジェクトフォルダに移動します。

```bash
cd blog
```

次に、依存パッケージをインストールします。

```bash
npm install
```

開発サーバー起動

以下のコマンドで開発サーバーを起動します。

```bash
npm start
```

サーバー起動後、ブラウザで以下のURLにアクセスしてください。

```
http://localhost:3000
```

ビルド

本番用ビルドを作成する場合は以下のコマンドを実行します。

```bash
npm run build
```
